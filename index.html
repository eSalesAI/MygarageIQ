GarageIQ — One-time setup
Connect a free Firebase database so your garage syncs automatically across your phone, tablet, and computer. Takes about 3 minutes.
1
Go to console.firebase.google.com and sign in with your Google account.
2
Click Create a project → name it garageiq → disable Google Analytics → click Create project.
3
In your project, click Build → Realtime Database → Create database → choose United States → select Start in test mode → click Enable.
4
You'll see a URL at the top of the database page that looks like this — copy it and paste below:
https://garageiq-xxxxx-default-rtdb.firebaseio.com
